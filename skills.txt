Программа курса:
Базовая теория: Что такое тестирование, багрепорты, документация, виды, методы, направления тестирования, SDLC, STLC,
HTTP: Методы запросов на сервер, коды ответов, Структуры HTTP запросов и ответов,
Форматы обмена данными: JSON, XML,
Тестирование API: Postman,
Снифинг http web траффика: Charles, Fiddler,
Dev Tools для тестировщиков: браузеры Google Chrome, FireFox,
Командная оболочка Bash (UNIX): Создание скриптов, работа с файлами, поиск (find, grep), система контроля версий Git,
Основы SQL: Create, Delete, Drop, Insert Into, Select, From, Where, Join
Нагрузочное тестирование: Jmeter
